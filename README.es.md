## Qué es WebMinutes

**WebMinutes** es una extensión ligera para Chrome cuya misión es ayudarte a **ser más consciente del tiempo que gastas online**, especialmente en redes sociales.  
- No requiere cuenta.  
- Los datos se guardan **en tu dispositivo** (privacidad primero).  
- UX minimalista + visual, sin complicaciones.

### Características (MVP)
- Conteo por **URL activa** + **detección de inactividad**.  
- Agrupación por apps (YouTube, Instagram, X, …) con favicons dinámicos.  
- Estadísticas del día: tiempo total, top páginas/apps.  
- Exportar CSV.  
- Reset del día.

Próximamente: histórico semanal/mensual, alertas inteligentes, dashboard visual y tema claro.

## Instalación (modo desarrollador)

```bash
git clone https://github.com/SentoMarcos/WebMinutes.git
cd WebMinutes
```

1. Abre `chrome://extensions` y activa **Modo desarrollador**.  
2. Clic en **Cargar descomprimida** y selecciona la carpeta del proyecto.  
3. Fija el icono de la extensión y abre el popup. Listo.
